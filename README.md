# Frontend MongoDB Lab 3

## https://mongodblab3.netlify.app/index.html

Webbplats för frontend lab 3. 
Skapad med vite, vanlig HTML, CSS och Javascript. DOM-manipulation och fetches till CRUD-API-länkar hos en backend webtjänst hanterad av Render, i sin tur av MongoDB Atlas.

Inbyggd funktionalitet för att läsa (GET) innehåll. Två undersidor för att skapa (POST) respektive uppdatera (PUT) poster. 
På finns alternativ för att uppdatera post (leder till undersida "add.html") och för att radera (DELETE) post.
